产品简介

对于每个管理子系统来讲，应用级的安全可以说是更关键的。应用级安全管理主要是指这种对于应用系统中某些功能的使用权限的控制。 登录口令、等敏感数据，在应用程序使用时必须进行加密存储和传输，避免在配置中出现明文口令等操作，保障系统应用级安全。 数据可靠性维护设计主要包括二个方面：录入错误检查，数据库数据一致性保证。 录入错误检查是在凡具有数据录入的程序模块内部将录入的数据进行合法性检查，合法性检查通过的数据在存储到数据库。关键数据项采用双次录入以减少录入错误。
